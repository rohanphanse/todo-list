# Todo List
A simple todo list created with React.js. This was inspired by Brad Traversy's React crash course tutorial.

## Get started
```JS
// Install dependencies
npm install

// Run on localhost:3000
npm start

// Build for production
npm build
```

## Features
* Manage Todo List
  - Add
  - Complete
  - Delete
* Custom CSS styling
* React.js
